# Solution to Rubik Cube Challenge

## Pre-requisites

* .NET Core 9 SDK

## How to build


### Build using IDE

* Open `RubikCube.sln` file in your favorite IDE (Visual Studio, Rider, etc.)
* Build the solution

### Build using CLI

* Run ```dotnet build``` in the root directory of the project

## How to run

* Run ```dotnet run``` in _Src/RubikCube_ directory
